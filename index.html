About Me

Hello, My name is Geoffrey Park, and welcome to my portfolio website. I am an aspiring game dev currently studying in the GIMM program at Boise State University, and this website was made to show off the projects I have created thus far. I have experience in game design, unity, 2d art and animation, UI/UX design, and some web design. I desire to work in a creative field like game development as I feel it will allow me to really express my artistic abilities, as well as allow me to further grow my skills.

Portfolio Artifacts

Artifact 1 - GIMM 100 Final Animation

This artifact is the final animation project I did for GIMM 100, a culmination of all of the work we had done for that class. During this project, I learned how to put together a proper animation along with sound effects, before uploading it to youtube. The thing that made this project fun to create for me was being able to go from a simple concept I had made at the start of the semester and make an entire animation out of it, seeing a design I had made early on moving was great fun.

Artifact 2 - GIMM 200 Final Animation

This artifact is my final animation that I created for my semester of GIMM 200. Within this project, I did more practice with illustration, animation, concepting, and video editing. The thing that was fun about this specific project was creating the design for the airship that was featured in the animation, as I had a lot of fun designing and animating it during its feature scene.

Artifact 3 - GIMM 200 UX Project

This artifact was my UI/UX project for GIMM 200 in which we designed the layout of some kind of app making use of Adobe XD. In this project I created a mock up of a Music app, in the process of creating it learning to make an easy to use layout similar to many others currently in use.
